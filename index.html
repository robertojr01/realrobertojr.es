2. Proyectos

Proyecto 1

Lectura Rápida

Consiste en ayudar a mejorar el nivel de lectura de cada persona, está basado en la aplicación de 3 principios básicos que están demostrados por estudios conseguir resultados a partir de las 2 semanas.
Esta web te acompaña en el proceso y te ayuda a conseguirlo.

El primer principio es utilizar la visión periférica para concentrarla en 3 o 4 puntos por línea, el segundo principio es eliminar la vocalización interior y el tercero es dominar la concentración a corto plazo, que es aquella que utilizamos cuando estudiamos o leemos.

Proyecto 2

Control de Horas

Es una aplicación web que se podrá instalar como aplicación de escritorio que tiene el fin de controlar o saber gestionar las horas del día, semana o mes.
Se podrán monotorizar las horas que dedicas a actividades como la programación, la lectura o la edición de vídeo. El objetivo es conseguir ser más productivo.
La novedad es que por cada objetivo diario o semanal cumplido crecerá un trozo de una montaña que cada vez se hará más grande si logras mantenerte enfocado, si rompes la cadena y pierdes el ritmo, no podrás seguir y tendrás que empezar de nuevo.
